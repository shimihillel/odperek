# עוד פרק אחד — v49 Dynamic Covers

מה חדש:
- 4 וריאציות של כריכות מעוצבות אוטומטיות באותה שפה.
- כריכה דינמית בכל מקום באפליקציה: בית, עדכון מהשטח, סיכום, מדף ופרטי ספר.
- שם הספר ושם הסופרת מוצגים על הכריכה עם שבירת שורות אוטומטית.
- אפשר להעלות תמונה מהגלריה ככריכה אישית לספר.
- תמונות כריכה נשמרות מקומית ונכללות בגיבוי JSON.
- תיקון עיגול התקדמות: רק אחוזים בתוך העיגול.
- כפתור “נטשתי” בעדכון מהשטח הוקטן לפעולה משנית.
