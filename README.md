# עוד פרק אחד v5

גרסה גדולה:
- עיצוב חדש: דרמה רומנטית כיפית עם אופי, בלי כתב נטוי.
- מסך בית חדש סביב “קוראת עכשיו”.
- “עדכון מהשטח” הפך למסך ייעודי, לא פופאפ.
- כרטיסים מתקפלים וזורמים:
  - איפה אני
  - מה הוייב?
  - מה בתכלס קורה
  - לגזור ולשמור
  - על מי מדובר
- השמירה לא מקפיצה מקום.
- “על מי מדובר” תומך בכמה דמויות + זכר/נקבה.
- מסך סיום ספר נקרא “מסכמת”.
- סדר הסיכום:
  דירוגים, שורה תחתונה, בשורה אחת, דמויות, רגעים מהשטח.
- כפתור סיום: “וזהו”.
