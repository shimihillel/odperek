# עוד פרק אחד — v50 Real Gallery Covers

- אין יותר כריכות גנריות/מאוירות כספר עצמו.
- כריכה היא תמונה אמיתית מהגלריה בלבד.
- שם הספר והמחברת מוצגים מחוץ לכריכה, כחלק מה־UI.
- הכריכה מוצגת כמלבן אנכי ביחס 2:3, כמו כריכת ספר אמיתית.
- כשאין תמונה, מופיע placeholder עדין עם אפשרות להוסיף כריכה.
- נשמרת תמיכה בגיבוי/שחזור עם תמונות הכריכה המקומיות.
- נשמרו כל הפיצ׳רים מ־v49: מדף, נטושים, עריכת ספר, עריכת רגעים, ניהול תגיות, סטטיסטיקות וגיבוי.
