# עוד פרק אחד v3

עדכון:
- הכפתור הראשי נקרא “עדכון מהשטח”.
- הפופאפ הפך ללוח עבודה זורם: אפשר לעדכן כמה דברים ברצף בלי לצאת ולהיכנס.
- טאבים: התקדמתי, מצב, פריקה, ציטוט, דמות, ציר.
- אחרי שמירה נשארים בתוך הפופאפ ורואים את ציר הקריאה.
- שינוי עיצובי כיפי יותר: דוח שטח, פתקים, קלפים, וייב שולחן קריאה.
