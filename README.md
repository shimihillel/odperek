# עוד פרק אחד — v68

גרסת ניקוי נוספת למסכי הבית והעדכון.

## מה השתנה
- בדף הבית עדכון מהיר של ההתקדמות הוא עכשיו ב־1% לכל כיוון במקום 5%.
- במסך "עדכון מהשטח" הוסרו ארבעת כרטיסי הסיכום העליונים שחזרו על מידע שמופיע בהמשך המסך.
- אזור "הספר הזה מזכיר לי את…" הועבר לתחתית מסך העדכון, אחרי "רגעים אחרונים מהשטח".
- שאר מבנה הבית, העדכון, המדף והסטטיסטיקה נשאר ללא שינוי.
